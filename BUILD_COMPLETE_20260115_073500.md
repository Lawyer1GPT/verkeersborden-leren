## Known Limitations

1. **Limited Image Coverage** (9.7%)
   - Only 18/186 signs have official images
   - Quiz limited to these 18 signs
   - Browse mode shows placeholder for others
   - **Solution**: Extract remaining images from wetten.overheid.nl

2. **External Image Dependencies**
   - Images loaded from wetten.overheid.nl
   - Requires internet connection
   - Subject to government site availability
   - **Solution**: Download and host locally

3. **No User Data Persistence**
   - Progress not saved between sessions
   - No user accounts
   - No score history
   - **Solution**: Add localStorage or backend
